Projeto 2

O App de Controle de Gastos é um projeto que consiste no desenvolvimento de uma aplicação onde o usuário pode registrar e acompanhar todas as suas movimentações financeiras, incluindo entradas e saídas de dinheiro. A proposta é criar uma versão simplificada de aplicativos como Mobills e Guiabolso, permitindo que o usuário organize melhor sua vida financeira. Cada transação registrada no sistema possui informações como nome, valor, tipo (entrada ou saída) e data. O aplicativo começa com funcionalidades básicas, como adicionar, listar e excluir transações, além de calcular automaticamente o saldo total. Em versões mais completas, é possível editar registros, filtrar por tipo de transação e armazenar os dados no navegador utilizando LocalStorage, além de exibir separadamente o total de ganhos e gastos. Para um nível mais avançado, o projeto pode incluir gráficos para análise financeira, categorização de despesas (como alimentação, transporte e lazer), sistema de login de usuários, integração com banco de dados para armazenamento online, exportação de relatórios em PDF ou Excel e até mesmo a implementação de modo escuro. Trata-se de um projeto bastante completo, que pode evoluir de algo simples para uma aplicação robusta, sendo ideal para aprendizado e também para compor um portfólio profissional.
